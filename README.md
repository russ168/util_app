A client for Elastic Search in Erlang.
======================================

* rebar or sinan
* uses lhttpc http library

QUICKSTART with rebar
--------------------

```shell
$ rebar get-deps
$ rebar compile
$ ./start-dev.sh
```

QUICKSTART with sinan
---------------------

```shell
$ agner install lhttpc
$ ./sinan build
$ ./start-dev.sh
```
